> Linear uses OAUTH2 and API_KEY for authentication. Please refer to the [authentication page](/docs/authentication) for more details.





## ⚒️ Actions

Linear has 3 actions which you can start using immediately

 _⚠️ All the actions are disabled by default, Please go to the Github connector page to enable the actions._


| Action Name            | Action ID                   | Description                                       |
| :--------------------- | :-------------------------- | :------------------------------------------------ |
| Create Issue | linear_create_linear_issue | Create a new issue in a Linear account |
| Get Projects | linear_list_linear_projects | Get the list of projects in a Linear account |
| Get Teams by Project | linear_list_linear_teams | Get the list of teams in a Linear account filtered by project ID |

## 🪝 Triggers

Linear triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
