## ‚öíÔ∏è Actions

Github has 8 actions which you can start using immediately

 _‚ö†Ô∏è All the actions are disabled by default, Please go to the Github connector page to enable the actions._


| Action Name            | Action ID                   | Description                                       |
| :--------------------- | :-------------------------- | :------------------------------------------------ |
| Create Issue | github_create_issue | Create a new issue in a repository. |
| Get Repository | github_list_github_repos | Get the list of repositories in a GitHub account |
| Star Repo | github_star_repo | Star a repository. |
| Get About Me | github_get_about_me | Get information about the owner account in GitHub. |
| Fetch README | github_fetch_readme | Fetch the README file from a GitHub repository. |
| Get Commits | github_get_commits | Get the last n commits with messages and descriptions from a GitHub repository. |
| Get Commits With Patch file for that commit | github_get_commits_with_code | Get the last n commits with messages and descriptions, patch file for that commit from a GitHub repository. |
| Get Patch For Commit | github_get_patch_for_commit | Get the patch file for a specific commit in a GitHub repository. |
