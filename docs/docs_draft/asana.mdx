> Asana uses OAUTH2 for authentication. Please refer to the [authentication page](/docs/authentication) for more details.





## ⚒️ Actions

Asana has 2 actions which you can start using immediately

 _⚠️ All the actions are disabled by default, Please go to the Github connector page to enable the actions._


| Action Name            | Action ID                   | Description                                       |
| :--------------------- | :-------------------------- | :------------------------------------------------ |
| Create Subtask | asana_create_subtask | Create a subtask under a specific task. |
| Get Subtasks | asana_get_subtasks | Get subtasks for a specific task. |

## 🪝 Triggers

Asana triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
